---
title: "Site Settings"
description: "Public read and admin write endpoints for stake site configuration."
---

The settings router exposes the stake portal's content management interface. `GET /settings` is public and used by the frontend to hydrate site-wide content such as the stake name, hero text, and sacrament times. `PUT /settings` and the image upload endpoint require the `MANAGE_SITE_SETTINGS` permission.

Base path: `/settings`

## Endpoints

| Method | Path | Auth | Description |
|--------|------|------|-------------|
| `GET` | `/settings` | None | Return current site settings |
| `PUT` | `/settings` | `MANAGE_SITE_SETTINGS` | Partially update site settings |
| `POST` | `/settings/upload/{image_type}` | `MANAGE_SITE_SETTINGS` | Upload a logo or hero image |

---

## GET /settings

Returns the current `SiteSettingsResponse` object. Called on page load by the frontend to populate the navbar, footer, hero section, and other site-wide content.

**Response 200 — SiteSettingsResponse:**

```json
{
  "stake_name": "Provo Utah North Stake",
  "stake_address": "1234 N University Ave, Provo, UT 84604",
  "contact_email": "clerk@provostake.org",
  "reply_to_email": "clerk@provostake.org",
  "hero_title": "Welcome to the Provo Utah North Stake",
  "hero_subtitle": "Serving families across north Provo and the surrounding area.",
  "sacrament_times": ["9:00am", "11:00am", "1:00pm"],
  "hidden_pages": ["sports"],
  "logo_url": "/api/settings/images/logo",
  "hero_image_url": "/api/settings/images/hero"
}
```

| Field | Type | Description |
|-------|------|-------------|
| `stake_name` | string | Full stake name, shown in the navbar and page titles |
| `stake_address` | string | Physical address, shown in the footer and contact page |
| `contact_email` | string | Public contact email displayed on the site |
| `reply_to_email` | string | Reply-to address on outgoing emails |
| `hero_title` | string | Main heading on the home page hero section |
| `hero_subtitle` | string | Supporting text below the hero title |
| `sacrament_times` | list[string] | List of sacrament meeting times (e.g., `"10:00am"`) |
| `hidden_pages` | list[string] | Page keys that are hidden from navigation |
| `logo_url` | string \| null | URL to the stake logo image, or `null` if not uploaded |
| `hero_image_url` | string \| null | URL to the hero background image, or `null` if not uploaded |

---

## PUT /settings

Partially updates site settings. Only the fields included in the request body are modified; omitted fields retain their current values.

**Request body (all fields optional):**

```json
{
  "stake_name": "Provo Utah North Stake",
  "stake_address": "1234 N University Ave, Provo, UT 84604",
  "contact_email": "clerk@provostate.org",
  "reply_to_email": "clerk@provostate.org",
  "hero_title": "Welcome to Our Stake",
  "hero_subtitle": "Find resources, schedules, and leadership information.",
  "sacrament_times": ["9:00am", "11:00am"],
  "hidden_pages": ["sports", "calendar"]
}
```

| Field | Type | Description |
|-------|------|-------------|
| `stake_name` | string | Full stake name |
| `stake_address` | string | Physical mailing or meeting address |
| `contact_email` | string | Public-facing contact email |
| `reply_to_email` | string | Reply-to address for outgoing emails |
| `hero_title` | string | Home page hero heading |
| `hero_subtitle` | string | Home page hero subheading |
| `sacrament_times` | list[string] | Sacrament meeting times; order determines display order |
| `hidden_pages` | list[string] | Page keys to hide from navigation (e.g., `"sports"`, `"calendar"`) |

**Response 200:** The full updated `SiteSettingsResponse` object.

---

## POST /settings/upload/{image_type}

Uploads a logo or hero background image. The file replaces any previously uploaded image of that type.

**Path parameter:**

| Parameter | Values | Description |
|-----------|--------|-------------|
| `image_type` | `logo` \| `hero` | Which image slot to update |

**Request:** `multipart/form-data` with a single `file` field.

**Accepted formats:** JPEG, PNG, WebP

**Maximum file size:** 5 MB

**Example (curl):**

```bash
curl -X POST http://localhost:3100/api/settings/upload/logo \
  -H "Authorization: Bearer <access_token>" \
  -F "file=@/path/to/stake-logo.png"
```

**Response 200:**

```json
{ "url": "/api/settings/images/logo" }
```

**Response 400:** File exceeds 5 MB, or the format is not accepted.

<Callout type="info">
  The `MANAGE_SITE_SETTINGS` permission must be explicitly assigned. It is not implied by other admin-level permissions such as `MANAGE_USERS` or `MANAGE_CALLINGS`.
</Callout>
