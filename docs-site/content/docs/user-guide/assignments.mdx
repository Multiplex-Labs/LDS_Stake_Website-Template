The Assignments page (`/leader/assignments`) is a **read-only** reference table that shows every High Councilor slot, who currently fills it, how to reach them, and what stake responsibilities and committee they hold. Editing is done separately in the [Admin Hub](/user-guide/admin).

<Callout type="info">
  Access to this page requires an authenticated leader account. The page is visible to all users who can log in to the Leader Portal.
</Callout>

## Viewing the Assignments Table

Navigate to `/leader/assignments`. The table contains one row per High Councilor slot and displays the following columns:

| Column | Description |
|--------|-------------|
| **High Councilor** | The full name of the person currently holding that slot |
| **Phone** | The High Councilor's phone number on file |
| **Responsibility** | One or more badges showing the stake assignment areas this person covers |
| **Committee** | A badge indicating the committee the High Councilor serves on |

Rows are generated automatically from the active High Councilor calling holders. If a slot is vacant or has no responsibility or committee information entered yet, those cells will appear empty.

## Editing Responsibility and Committee (Admin Hub)

Authorized administrators can update the **Responsibility** and **Committee** fields for each High Councilor slot. This is done in the Admin Hub, not on the main Assignments page.

<Callout type="warn">
  Editing assignment details requires the **MANAGE_ASSIGNMENTS** permission. If you do not see the Admin Hub or the edit controls described below, contact your stake clerk or system administrator.
</Callout>

<Steps>
  <Step title="Open the Admin Hub">
    Navigate to `/leader/admin` and select the **HC Assignments** tab, or use the direct link `/leader/admin?tab=hc-assignments`.
  </Step>
  <Step title="Locate the slot to edit">
    Find the row for the High Councilor slot you want to update. Each row shows the slot number and the current occupant's name.
  </Step>
  <Step title="Click the Edit button">
    Click the **pencil (Edit)** icon on the right side of the row. A dialog opens with two editable fields.
  </Step>
  <Step title="Update the fields">
    - **Responsibility** — Enter the stake assignment areas this High Councilor covers. You can add multiple values as separate tags.
    - **Committee** — Enter the committee name this High Councilor serves on.
  </Step>
  <Step title="Save your changes">
    Click **Save** in the dialog. The updated values appear immediately on both the Admin Hub table and the main Assignments page.
  </Step>
</Steps>

## Ward-to-Slot Assignments

The association between a High Councilor slot and the ward(s) that slot covers is **not managed through the portal UI**. Ward assignments are configured by your system administrator in the `wards.csv` file and through the calling slot structure set up when the application is first deployed.

<Callout type="info">
  If a ward assignment needs to change — for example, a different High Councilor slot should cover a given ward — contact your system administrator. They will update the configuration and restart the backend service for the change to take effect.
</Callout>
