This guide introduces the Stake Portal's layout, explains how to log in, and maps each Leader Portal section to its documentation page.

## Logging In

Navigate to `/login` and enter your **email address** and **password**, then click **Sign In**.

<Callout type="note">
  If your account was created by an administrator, you will be redirected to `/change-password` immediately after your first successful login. You must set a new password before you can access any portal features. This is a one-time requirement.
</Callout>

## The Navigation Bar

The top navigation bar is present on every page.

- **Left side** — the portal logo and site name; clicking it returns you to the home page.
- **Right side** — a profile menu showing your name and email address. Click it to access your profile settings or to **log out**.
- **Dark mode toggle** — also in the top-right area; switches between light and dark themes. Your preference is remembered across sessions.

All leader-only pages live under the `/leader` prefix and require you to be authenticated. Use the navigation menu or the cards below to jump to any section.

<Callout type="info">

</Callout>

## Admin Hub Tabs

The Admin Hub at `/leader/admin` is the primary editing interface for stake data. It is organized into the following tabs:

| Tab | URL | What you can manage |
|-----|-----|---------------------|
| Users | `/leader/admin?tab=users` | Create, edit, and deactivate leader accounts; assign callings and upload photos |
| Callings | `/leader/admin?tab=callings` | Define calling names, slot counts, and visibility settings |
| Wards | `/leader/admin?tab=wards` | Edit ward names and sacrament meeting start times |
| Speaking Assignments | `/leader/admin?tab=speaking` | Assign wards to HC slots by month; edit monthly speaking topics |
| High Council Assignments | `/leader/admin?tab=hc-assignments` | Edit each slot's responsibility text and committee label |
| Presidency Assignments | `/leader/admin?tab=presidency` | Manage Stake Presidency member assignments |
| Appointment Manager | `/leader/admin?tab=temple-recommend` | Manage Temple Recommend appointment types and interviewer availability (visible only to users with the MANAGE_APPOINTMENTS permission) |

<Callout type="note">
  The Appointment Manager tab only appears if your account has the MANAGE_APPOINTMENTS permission (bitmask value 512). If you do not see it, contact your system administrator.
</Callout>

## Calling Pipeline at a Glance

The calling proposal workflow follows a structured Kanban pipeline. Here is a quick summary of the stages a new calling passes through:

| Stage | Who Acts |
|-------|----------|
| SP Approval | Stake Presidency members approve the proposal |
| HC Approval | High Councilors approve the proposal |
| Interview | Candidate is interviewed by SP or HC |
| Sustain | Candidate is sustained in sacrament meeting |
| Set Apart | Person is set apart in their new calling |
| LCR Update | Calling is recorded in Leader and Clerk Resources |
| Done | Process is complete; proposal moves to the archive |

<Callout type="info">
  Releases (removals from a calling) skip the SP Approval and HC Approval stages and begin at Interview. They also skip the Set Apart stage.
</Callout>

See the [Callings section](/user-guide/callings/submit) for full details on submitting and managing proposals.
