## Accessing the Page

Navigate to `/leader/sustainings`. This page provides the verbatim language a bishop or presiding officer reads when sustaining or releasing members in a sacrament meeting.

## Ward Tabs

At the top of the page is a row of tabs — one tab per ward in the stake. Click a ward's tab to switch to that ward's sustaining content.

Each tab displays the formatted sustaining and release language specific to that ward. You can navigate freely between wards without losing your place.

## Reading the Language

The text displayed under each ward tab is the exact, verbatim language to be read aloud. It is formatted for ease of reading in a sacrament meeting setting. No editing or reformatting is needed before use.

<Callout type="info">
  The sustaining language follows the format used by The Church of Jesus Christ of Latter-day Saints for conducting sustainings in sacrament meetings. It includes the names of those being called or released and the appropriate congregational response prompt.
</Callout>

## Current Limitations

<Callout type="warn">
  The management interface for assigning names to the sustaining list is not yet available in the portal. The viewer displays sustaining language based on data currently in the system, but the ability to add, edit, or remove names that appear in the verbatim text is future work. This feature is planned for a future release.
</Callout>

At this time:

- You **can** view the sustaining language for each ward.
- You **cannot** directly manage which names appear in the sustaining text from within the portal.

For questions about updating the names shown in the sustaining language, contact your system administrator or stake clerk.
