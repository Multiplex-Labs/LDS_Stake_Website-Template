# Quickstart

This guide gets you from zero to a running local instance. For production deployment, see [Docker Compose](/deployment/docker) or [Manual deployment](/deployment/manual).

## Prerequisites

- Node.js 18 or later
- Python 3.11 or later
- [uv](https://docs.astral.sh/uv/) (Python package manager)
- Git

## 1. Clone the repository

```bash
git clone <repository-url>
cd LDS_Stake_Website-Template-2
```

## 2. Set up the backend

```bash
cd backend
uv sync
```

Create a `.env` file:

```bash
cat > .env << 'EOF'
DEV=true
PORT=8000
DB_ENGINE=sqlite
DATABASE_PATH=./database.db
INITIAL_ADMIN_PASSWORD=admin123
JWT_SECRET_KEY=change-me-to-a-random-secret
SSL_ENABLED=false
SP_APPROVAL_THRESHOLD=2
HC_APPROVAL_THRESHOLD=3
EOF
```

Start the backend (first run seeds the admin account):

```bash
uv run python main.py
```

<Info>
On first launch, the backend creates the `admin@admin.com` account with your `INITIAL_ADMIN_PASSWORD` and loads ward data from `wards.csv`. You will be prompted to change the password on first login.
</Info>

## 3. Set up the frontend

Open a new terminal:

```bash
cd frontend
npm install
npm run dev
```

## 4. Open the portal

Navigate to [http://localhost:3100](http://localhost:3100) in your browser.

Log in with:
- **Email:** `admin@admin.com`
- **Password:** the value you set for `INITIAL_ADMIN_PASSWORD`

You will be redirected to change your password before accessing the portal.
