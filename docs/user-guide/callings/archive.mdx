## Accessing the Page

Navigate to `/leader/callings/archive`. The archive is a read-only historical record of every calling proposal that has reached the **Done** stage.

## What Appears Here

Only proposals that have completed the full pipeline and reached the **Done** stage are shown in the archive. Active proposals (those still in progress on the Kanban board) do not appear here.

<Info>
  A proposal moves to the archive automatically when it is advanced to Done from the LCR Update stage on the [Manage (Kanban)](/user-guide/callings/manage) page.
</Info>

## Searching the Archive

Use the **search bar** at the top of the page to filter proposals by:

- Candidate name (first or last name)
- Calling title
- Ward name

The list filters as you type — no need to press Enter.

## Filtering

In addition to the search bar, you can apply filters to narrow the archive:

| Filter | Options |
|--------|---------|
| Ward | Select a specific ward to show only proposals from that ward |
| Type | Filter by **Calling** (new callings) or **Release** to view each category separately |

Filters can be combined with the search bar.

## Reading an Archived Proposal

Click any row in the archive table to open the full proposal detail view. You will see:

- **Candidate name** — first and last name
- **Proposed calling** — the calling title
- **Ward** — the ward this calling belonged to
- **Submitter** — who submitted the proposal
- **Spouse name** — if provided
- **Is Release** — whether this was a new calling or a release
- **Submission date** — when the proposal was originally created

### Stage History

Below the proposal details, the full **stage history** is displayed. This shows every stage transition — who moved the proposal, from which stage, to which stage, and at what date and time. This provides a complete audit trail of how long each stage took and who was involved.

## Read-Only

Archived proposals cannot be edited, commented on, or moved back into the active pipeline. If a correction is needed for a completed proposal, submit a new proposal with the corrected information.

<Warning>
  There is no way to delete an archived proposal from the portal UI. The archive is a permanent record.
</Warning>
